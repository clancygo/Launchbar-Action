# LaunchBar Actions

Personal backup of custom LaunchBar actions.

## Installation

Double-click an `.lbaction` bundle in Finder, or copy it to:

`~/Library/Application Support/LaunchBar/Actions/`

## Actions

- **Finder Tabs**: lists open Finder windows and focuses the selected window.
- **提取 PDF 页码**: send a PDF to the action, enter a range such as `12-27` or `1-3,8,10-12`, and save the extracted PDF beside the source file.
